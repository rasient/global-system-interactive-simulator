# Model architecture

## Core state variables

- Incentive alignment
- Verifiable trust
- Shared reality
- Adaptation speed
- Infrastructure resilience
- Reinforcement quality
- Local-global balance
- Complexity management

## External pressures

- AI pressure
- Geopolitical pressure
- Resource pressure
- Social pressure

## Main output metrics

- System health
- Pressure
- Fragility gap
- Coordination capacity
- Resilience
- Agent outcomes
- Risk profile

## Simulation logic

Each year:

1. System health is calculated from weighted design variables.
2. Pressure is compared against health.
3. Fragility gap is calculated.
4. Interventions improve selected dimensions.
5. Pressure damages weak dimensions.
6. Reinforcement loops amplify or weaken future adaptation.
7. Trust and shared reality improve coordination capacity.
8. The model classifies the trajectory.

## Important note

This is not a scientific forecasting model.

It is a strategic thinking simulator.
