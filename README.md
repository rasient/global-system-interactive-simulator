# Global System Interactive Simulator

A Streamlit-based interactive simulator for exploring how global systems evolve under pressure.

It models:

- AI + labor disruption
- infrastructure stress
- geopolitical fragmentation
- social reinforcement loops
- trust and shared reality
- incentive alignment
- adaptation speed
- systems-of-systems complexity

## Why this exists

Many systems look well-designed on paper.

But they fail because reality is:

- noisy
- unpredictable
- full of hidden dependencies
- shaped by incentives
- constrained by infrastructure
- slowed by institutions
- reinforced by behavior loops

This simulator turns those ideas into an interactive prototype.

## Main features

- Scenario selector
- Region baseline selector
- Sector focus selector
- System-design sliders
- Multi-year simulation
- Fragility gap tracking
- Agent outcome model
- Risk diagnosis
- Intervention recommendations
- CSV export
- Markdown report export

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
